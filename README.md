## Lucky Number Test Game

This is a **Random Number Game** where multiple players can participate. The main objective of the game is to win or lose based on a randomly generated number. This project is built using **HTML, CSS, JavaScript, and GSAP**.

---

## How the Game Works

1. **Create a Player**

   * The player enters their **name** and the **amount of money** they want to bet.
   * After clicking the `Create Player` button, a **player box** is created.

2. **Select a Number**

   * Each player selects **one number** from their player box.
   * After selecting a number, the player clicks the `Update Number` button.
   * ‚ö†Ô∏è If a player does **not select a number** and chooses something else, the **loss is confirmed**.

3. **Check the Result**

   * After all players have selected their numbers, click the `Check Result` button.
   * Each player‚Äôs **result (Win / Lose)** is displayed in the **Result Box**.
   * The result is decided based on a **random number generation**.

## üìú Game Rules

1. Each player must enter a **valid name** and **bet amount** to join the game.
2. A player is created only after clicking the **Create Player** button.
3. Every player must **select exactly one number**.
4. After selecting a number, the player must click the **Update Number** button to confirm their choice.
5. ‚ö†Ô∏è If a player clicks **Update Number** without selecting a number, the player will **automatically lose**.
6. The game result is calculated using a **randomly generated number**.
7. When the **Check Result** button is clicked, the result for **all players** is displayed in the Result Box.
8. Once the result is shown, the current round ends.
